# 🚦 Smart Traffic Management System (AI-Based)

## 📌 Problem
Traditional traffic lights use fixed timing, leading to congestion, delays, and inefficient traffic flow.

## 💡 Solution
This project simulates an AI-based smart traffic system that dynamically adjusts signal timings based on real-time traffic density.

## ⚙️ Features
- 🚗 Real-time traffic simulation
- 🤖 AI-based decision making
- ⏱️ Dynamic signal timing (NS vs EW)
- 📊 Live decision logs
- 💻 Interactive dashboard UI

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js (optional simulation)
- Logic: JavaScript (AI rule-based)

## ▶️ How to Run

### Client
cd client
npm install
npm run dev

### Server (if used)
cd server
npm install
node index.js
